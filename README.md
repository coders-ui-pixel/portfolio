# Bal Krishna — Modern Personal Platform

## 🚀 Overview
A premium, production-ready personal portfolio and management platform. Fully customizable by admin, featuring modern glassmorphism aesthetics and advanced interactivity.

## ✨ New Features
- **🌓 Dark/Light Theme**: Smooth theme switching with local persistence.
- **⌨️ Typing Effect**: Animated typing effect on the hero subtitle (customizable speed).
- **🔒 Hardened Security**: All passwords are now hashed using **SHA-256** (Web Crypto API).
- **📐 Full Admin Control**: Customize Header, Footer, Hero, Skills, Projects, and Social links directly from the dashboard.
- **🎨 Modern UI**: Built with Glassmorphism, Syne & Space Grotesk typography, and smooth micro-animations.
- **📱 Responsive**: Optimized for all devices with a mobile-friendly layout.
- **📊 Excel Integration**: Export user data and records to .xlsx format instantly.

## 📦 Tech Stack
- Pure HTML5 + CSS3 (Modern Variables & Backdrop Blur)
- Vanilla JavaScript (Async/Await, Web Crypto API)
- SheetJS (Excel Export)
- Google Fonts (Space Grotesk, Syne, JetBrains Mono)
- FontAwesome 6 (Icons)
- localStorage (Zero-backend data persistence)

## 🛠️ Production Ready
- SEO Optimized with Meta Tags and Open Graph.
- Sanitized user inputs to prevent XSS.
- Fast loading and fully self-contained in a single file.
